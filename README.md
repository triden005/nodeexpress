# KnowledgeBase
----
This is a Basic app using Node.js & Express
### Technologies
  * Node.js
  * Express
  * Express Messages,Session ,Connect Flash & Validation
  * MongoDB & Mongoose
  * Pug Templating
  * Passport.js Authentication
  * Bcrypt Hashing

### Version
    1.0.0

## Usage
### Installation
    Install the dependencies
    using npm install
    &     npm start
